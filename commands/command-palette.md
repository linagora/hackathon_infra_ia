# Command Palette

Quick access to built-in commands, custom commands and skills.

## Access

- `Ctrl+P`

## Shortcuts

Type to filter the command list.

| Shortcut | Action                   |
| -------- | ------------------------ |
| `Enter`  | Execute selected command |
| `Escape` | Close dialog             |

## Main Commands

| Command     | Aliases                | Description                 |
| ----------- | ---------------------- | --------------------------- |
| `/new`      | `/clear`               | New session                 |
| `/sessions` | `/resume`, `/continue` | Switch sessions             |
| `/models`   |                        | Switch model                |
| `/agents`   |                        | Switch agent                |
| `/compact`  | `/summarize`           | Compact / summarize session |
| `/undo`     |                        | Undo previous message       |
| `/redo`     |                        | Redo undone message         |
| `/skills`   |                        | Browse skills               |
| `/editor`   |                        | Open external editor        |
| `/themes`   |                        | Switch theme                |
| `/help`     |                        | Show help                   |
| `/exit`     | `/quit`, `/q`          | Exit the app                |
