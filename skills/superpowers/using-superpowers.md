# using-superpowers

> Use when starting any conversation -- establishes how to find and use skills.

The meta-skill. Even a 1% chance a skill applies means invoke it.

## Rule

Invoke the skill tool **before any response**, including clarifying questions.

## Instruction Priority

1. User instructions (highest)
2. Superpowers skills
3. Default system prompt (lowest)

## Skill Types

- **Rigid** -- must be followed exactly (e.g., verification-before-completion)
- **Flexible** -- adapt to context (e.g., brainstorming)

## Skill Priority

Process skills first (brainstorming, writing-plans), then implementation skills (TDD, debugging).

## Red Flags

- "This is just a simple question" -- still check skills
- "I need more context first" -- load the skill first, then ask
- "I'll check skills after I respond" -- too late
