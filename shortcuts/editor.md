# Editor Shortcuts

Shortcuts for the message input area.

## Submission

| Shortcut                                                 | Action         |
| -------------------------------------------------------- | -------------- |
| `Return`                                                 | Submit message |
| `Shift+Return` / `Ctrl+Return` / `Alt+Return` / `Ctrl+J` | Insert newline |
| `Ctrl+C`                                                 | Clear input    |
| `Ctrl+V`                                                 | Paste          |

## Cursor Movement

| Shortcut                             | Action          |
| ------------------------------------ | --------------- |
| `Left` / `Ctrl+B`                    | Move left       |
| `Right` / `Ctrl+F`                   | Move right      |
| `Up`                                 | Move up         |
| `Down`                               | Move down       |
| `Ctrl+A`                             | Start of line   |
| `Ctrl+E`                             | End of line     |
| `Alt+F` / `Alt+Right` / `Ctrl+Right` | Word forward    |
| `Alt+B` / `Alt+Left` / `Ctrl+Left`   | Word backward   |
| `Home`                               | Start of buffer |
| `End`                                | End of buffer   |

## Deletion

| Shortcut                                      | Action                  |
| --------------------------------------------- | ----------------------- |
| `Backspace`                                   | Delete character before |
| `Ctrl+D` / `Delete`                           | Delete character after  |
| `Ctrl+K`                                      | Delete to end of line   |
| `Ctrl+U`                                      | Delete to start of line |
| `Ctrl+Shift+D`                                | Delete entire line      |
| `Alt+D` / `Alt+Delete` / `Ctrl+Delete`        | Delete word forward     |
| `Ctrl+W` / `Ctrl+Backspace` / `Alt+Backspace` | Delete word backward    |

## Undo / Redo

| Shortcut                   | Action |
| -------------------------- | ------ |
| `Ctrl+-` / `Super+Z`       | Undo   |
| `Ctrl+.` / `Super+Shift+Z` | Redo   |

## Input History

| Shortcut | Action                |
| -------- | --------------------- |
| `Up`     | Previous history item |
| `Down`   | Next history item     |
