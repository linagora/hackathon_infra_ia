# Model Selection Dialog

Allows changing the AI model used for the conversation.

## Access

- `<leader>m` (default: `Ctrl+X` then `m`)
- Or via command palette: `/models`

## Shortcuts

| Shortcut | Action                |
| -------- | --------------------- |
| `Ctrl+A` | Open provider list    |
| `Ctrl+F` | Toggle model favorite |
| `Escape` | Close dialog          |

## Quick Cycling (from chat)

| Shortcut   | Action                         |
| ---------- | ------------------------------ |
| `F2`       | Cycle to next recent model     |
| `Shift+F2` | Cycle to previous recent model |
| `Ctrl+T`   | Cycle model variants           |
