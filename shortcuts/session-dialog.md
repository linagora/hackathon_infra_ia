# Session Dialog

Session switcher for navigating between existing conversations.

## Access

- `<leader>l` (default: `Ctrl+X` then `l`)

## Shortcuts

| Shortcut | Action         |
| -------- | -------------- |
| `Ctrl+R` | Rename session |
| `Ctrl+D` | Delete session |
| `Escape` | Close dialog   |
