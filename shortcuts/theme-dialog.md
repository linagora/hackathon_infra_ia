# Theme Dialog

Allows changing the visual theme of the interface.

## Access

- `<leader>t` (default: `Ctrl+X` then `t`)
- Or via command palette: `/themes`

## Shortcuts

| Shortcut | Action       |
| -------- | ------------ |
| `Enter`  | Apply theme  |
| `Escape` | Close dialog |

## Built-in Themes (33)

aura, ayu, carbonfox, catppuccin, catppuccin-frappe, catppuccin-macchiato, cobalt2, cursor, dracula, everforest, flexoki, github, gruvbox, kanagawa, lucent-orng, material, matrix, mercury, monokai, nightowl, nord, one-dark, opencode, orng, osaka-jade, palenight, rosepine, solarized, synthwave84, tokyonight, vercel, vesper, zenburn

A **system** theme is also available, dynamically generated from the terminal's color palette.

Custom themes can be added via `themes/*.json` files in the config directory.
